# SwiftUIAlarm

SwiftUIAlarm is an example Alarm-like application using [SwiftUI](https://developer.apple.com/xcode/swiftui/) which is first introduced in WWDC19 keynote.



![screenshot](./README/SwiftUIAlarm-Screenshot.png)

## Requirements

* Xcode 11 Beta
* Swift 5.1
